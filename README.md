Бот для поиска пользователей Telegram по окончанию ID в публичных чатах и каналах.

- Поиск пользователей по 3-значному окончанию ID
- Поддержка множества чатов (до 100 за запрос)
- Использование сессий для асинхронного парсинга
- Все ответы в формате JSON
- Работа только с открытыми источниками


Бот модет работать без сессий, но так же поддерживает многопоточность. Заполняете файл config.py, после чего запускаете main.py. Заполняете, если нужно, свою базу чатов, где будут парсится люди - chats.csv, после чего можете пользоваться ботом.
